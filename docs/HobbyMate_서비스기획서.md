# 서비스 소개

HobbyMate는 같은 취미를 가진 사람들이 안전하게 모임을 만들고,
온라인/오프라인 만남까지 이어질 수 있는 취미 모임 플랫폼이다.

---

# 핵심 기능

## 회원

- 일반 회원가입
- 휴대폰 본인인증
- 성별 / 생년월일 검증
- 로그인
- 아이디 / 비밀번호 찾기
- 카카오 로그인 (추후 추가 예정)
- 관리자 / 일반회원 권한 분리

---

## 모임

- 모임 생성
- 모임 검색
- 카테고리별 조회
- 모임 가입
- 승인제 가입
- 성별 제한
    - 여성만
    - 남성만
    - 혼성
- 연령 제한
- 모집 인원 제한

---

## 모임 내부

모임마다 독립적인 공간을 가진다.

- 모임 소개
- 가입하기
- 자유게시판
- 만남 모집 게시판
- 만남 후기 게시판

모든 내부 게시판은 **모임 가입자만 접근 가능**하다.

---

## 게시판

### 공통

- 블로그형 에디터
- 글 중간 이미지 삽입
- 이미지 최대 25장
- 이미지 정렬
    - 왼쪽
    - 가운데
    - 오른쪽

### 전체 게시판

- 공지사항
- 자유게시판

### 건의 / 신고

- 작성자 본인과 관리자만 조회 가능
- 관리자 답변
- 처리 상태 관리

---

## 만남

### 대면

- 장소 검색
- 지도 API
- 카카오맵 또는 네이버지도

### 비대면

- Zoom
- Google Meet
- 기타 화상회의 링크

# 관리자

- 회원 관리

- 신고 관리
- 건의 관리

## 관리자 처리 이력

주요 관리자 작업은 모두 감사 이력으로 저장한다.

- 회원 정지
- 회원 정지 해제
- 모임 차단
- 게시글 차단
- 댓글 차단
- 신고 처리
- 건의 처리
- 데이터 복원

테이블

```text
HM_ADMIN_ACTION_HISTORY
```
